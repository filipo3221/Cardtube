# READ ME!
You will need an **SD** card for it to work! So grab a one yourself.

# Preparing the cardputer
At first, you will need a cardputer (ADV works the best for this) and an **SD** card formated to **FAT32**.
Download the latest **.bin** file from my github (or use the **CardTube** version from **M5Burner** and burn it)
Then, you need to launch the **.bin** file from Launcher on your cardputer.

# How to download videos
In **CardTube**, you can download videos.
1. Ensure you are connected to **Wi-Fi** in the **[SETTINGS]** tab.
This will allow you to **download** videos.

2. Go to the **[ONLINE]** tab and select a video from the list.

3. **Wait** for the download to complete.
**That's how you download videos and play them!**

# Watching offline
- Once downloaded, switch to the [SD CARD] tab. Your videos are stored locally and can be played anytime without an internet connection!
> ## **IMPORTANT:** Currently, sound (**.wav** files) is under development. Downloaded videos will play without audio for now.
