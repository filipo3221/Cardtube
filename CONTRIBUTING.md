# Contributing to CardTube

Thank you for your interest in improving CardTube! Since the source code is currently private, here is how you can help:

## 📢 How to Contribute
* **Bug Reports:** If the app crashes or doesn't work as expected, please open an "Issue" and describe your hardware (Cardputer or Cardputer ADV).
* **Feature Requests:** Have a cool idea? Let me know in the Issues section!
* **Video Suggestions:** If you want to see specific videos in the [ONLINE] tab, feel free to suggest them.

## 💬 Community
If you want to talk about the project or need quick help, find me on Discord!
